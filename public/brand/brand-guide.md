# ALAI Brand Guide

## Logo Usage

### Primary Logo Mark (DEFAULT — use everywhere)
**"A" + green dot** (`logo-icon.svg` / `profile-pic.png`) — The ALAI brand mark.
Use this as the primary logo on ALL platforms: social media, avatars, favicons, app icons, documents, presentations. This is the face of ALAI.

### Secondary: Wordmark (when space allows)
**"ALAI" + green dot on "i"** (`logo-full.svg`) — Extended wordmark.
Use alongside the logo mark when you need the full company name: website headers, document headers, presentations title slides, business cards.

### Tagline (NOT part of logo — use separately)
**"Norwegian AI Tech Holding Company"** — Use on website hero, LinkedIn about, presentation subtitles. Never lock it to the logo.

### Variations
| File | Type | Usage |
|------|------|-------|
| `logo-icon.svg` | **Primary mark** | ALL platforms, avatars, favicons, icons |
| `profile-pic.png` | **Primary mark (raster)** | Social media profiles, where PNG required |
| `logo-full.svg` | Wordmark (white) | Dark backgrounds — website, docs |
| `logo-dark.svg` | Wordmark (dark) | Light backgrounds — print, white paper |
| `logo-light.svg` | Wordmark (white) | Dark backgrounds — same as full |

### Clear Space
Maintain minimum clear space equal to the height of the "A" around all sides of the logo.

### Minimum Sizes
- Logo mark (A + dot): 32px minimum
- Wordmark (ALAI): 100px width minimum

### Don'ts
- Never change the green dot color (#00E5A0)
- Never stretch or distort proportions
- Never add effects (shadows, gradients, glows)
- Never place on low-contrast backgrounds
- Never rotate or skew the logo
- Never use colors outside brand palette

## Typography

```
-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Oxygen', 'Ubuntu', 'Cantarell', 'Fira Sans', 'Droid Sans', 'Helvetica Neue', sans-serif

```

```
H3:       Semibold, 36px, normal tracking, 1.3 line-height
H4:       Medium, 24px, normal tracking, 1.4 line-height
H5:       Medium, 20px, normal tracking, 1.5 line-height

Small:    Regular, 14px, normal tracking, 1.5 line-height

```

### Code Font
**JetBrains Mono** or system monospace for code blocks and technical content.

## Voice & Tone

### Writing Style
- Confident without arrogance
- Minimal without being cryptic
- Forward-looking without hype
- Technical when needed, clear always

### Do
- Use active, direct language
- Lead with insight, not buzzwords
- Be precise and specific
- Own the AI-native positioning
- Speak to decision-makers

### Don't
- Use AI hype or trendy phrases
- Over-explain or apologize
- Use corporate speak or jargon
- Make unsubstantiated claims
- Talk down to the audience

### Example Phrases
- "AI-native infrastructure for modern business"
- "Built for tomorrow, delivered today"
- "Technology that thinks ahead"
- "Precision-engineered solutions"

## Applications

### Business Cards
- Size: 85 x 55 mm (EU standard)
- Dark background (#09090b)
- Logo: icon variant, top left
- Green accent for contact details
- Paper: 400gsm, matte black with spot UV on logo

### Email Signature
```

```

### Presentations
- Dark backgrounds (#09090b) with generous whitespace
- Green accent (#00E5A0) for key points and CTAs
- One concept per slide
- Use logo-icon in footer or watermark
- Minimal text, maximum impact

### Website
- Premium dark-mode first design
- Green accent for interactive elements
- Smooth transitions and micro-interactions
- Focus on typography hierarchy
- Mobile-optimized from the start

### Social Media
- LinkedIn: Industry insights, thought leadership, company updates
- Twitter/X: Tech commentary, AI developments, fast takes
- Consistent dark-mode aesthetic across platforms
- Green accent in profile images and highlights
